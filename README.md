# Orenji Fitness

Orenji Fitness é uma aplicação web simples para organização de treinos, acompanhamento de progresso e personalização da experiência do utilizador.

A aplicação faz parte do Orenji Project, um ecossistema de apps web desenvolvido para demonstrar competências em frontend, UI/UX, persistência local e organização de código no contexto da PAF.

## Versão atual

**v1.0.0**  
Última atualização: **9 de junho de 2026**

## Funcionalidades

- Página inicial com apresentação da aplicação
- Dashboard com saudação personalizada
- Agenda semanal de treinos
- Catálogo de treinos por categoria
- Treinos de força, cardio, mobilidade, alongamento e HIIT
- Registo de séries, repetições, peso e duração
- Histórico de treinos concluídos
- Resumo de desempenho
- Cálculo de conclusão semanal
- Mapa anatómico de foco muscular
- Comparação de evolução por carga
- Sistema de temas com presets visuais
- Personalização avançada de cores
- Preferências de utilizador
- Exportação e importação de dados em JSON
- Limpeza de dados locais
- Persistência com localStorage

## Tecnologias utilizadas

- HTML5
- CSS3
- JavaScript Vanilla
- localStorage
- JSON
